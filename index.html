Frequently Asked Questions

Kya SparkInvoice really free hai?

Haan! Basic features completely free hain. Aap unlimited invoices bina kisi cost ke bana sakte hain. Premium features ke liye affordable plans available hain.

Voice recognition kitne languages mein kaam karta hai?

Hum Hindi aur English dono languages support karte hain. Aap mixing bhi kar sakte hain - "Item laptop add karo" ya "Add laptop" dono chalega!

Kya GST calculations automatic hote hain?

Bilkul! Aap bas GST percentage daalo, baaki calculation automatically ho jayega. Aap voice command se bhi GST on/off kar sakte hain.

Mobile app available hai kya?

Website fully mobile responsive hai. Aap kisi bhi browser mein use kar sakte hain. Progressive Web App (PWA) support hai - home screen pe add kar ke use karo!
